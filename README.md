## 🚀 Quick Start

### Option 1: Local Setup (Recommended)

1. **Download Files**
   ```
   - portfolio.html
   - Pratap_Choudhary_CV.pdf
   ```

2. **Place in Same Folder**
   ```
   my-portfolio/
   ├── portfolio.html
   └── Pratap_Choudhary_CV.pdf
   ```

3. **Open in Browser**
   - Double-click `portfolio.html`
   - Or right-click → Open with → Your Browser

### Option 2: Deploy to Web

#### GitHub Pages
1. Create a new repository on GitHub
2. Upload both files
3. Rename `portfolio.html` to `index.html`
4. Enable GitHub Pages in repository settings
5. Your site will be live at: `https://username.github.io/repository-name`

#### Netlify (Drag & Drop)
1. Go to [netlify.com](https://netlify.com)
2. Drag both files into Netlify Drop
3. Get instant live URL

#### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Navigate to folder with both files
3. Run: `vercel`
4. Follow prompts for deployment

## 🎨 Customization

### Change Colors
Edit the CSS variables in `portfolio.html`:

```css
:root {
    --bg-primary: #0f0f23;      /* Main background */
    --bg-secondary: #1a1a35;    /* Section background */
    --accent-primary: #6366f1;  /* Primary accent (indigo) */
    --accent-secondary: #ec4899; /* Secondary accent (pink) */
    --text-primary: #ffffff;    /* Main text color */
    --text-secondary: #94a3b8;  /* Secondary text */
}
```

### Update Content
All content is in `portfolio.html`:
- Line ~845-860: Stats numbers
- Line ~865-980: Skills descriptions
- Line ~985-1015: Certifications
- Line ~1020-1200: Projects
- Line ~1250-1280: Contact information

## 🔧 Troubleshooting

### CV Download Not Working
**Problem:** Download button doesn't work  
**Solution:** Both `portfolio.html` and `Pratap_Choudhary_CV.pdf` must be in the same folder

### Particle Background Not Showing
**Problem:** No particles visible  
**Solution:** Check internet connection (Particles.js loads from CDN)

### Animations Not Working
**Problem:** Elements don't animate on scroll  
**Solution:** Enable JavaScript in browser settings

### Theme Toggle Not Working
**Problem:** Can't switch between light/dark mode  
**Solution:** Clear browser cache and reload page
